# Linux Commands

## echo

'''bash
echo
'''

display, or "print," text directly onto your terminal screen.

## Notes

- Linux is case sensitive.
- Spaces between bytes matter

---

## whoami

'''bash
whoami
'''
current user session you are logged into and prints the associated username to the screen.

---

## id

'''bash
id
'''

to see what groups you belong to

## Notes

- uid: Your User ID (a unique numerical identifier).
- gid: Your primary Group ID.
- groups: All the groups you are a member of.

---


## id -un

'''bash
id -un
'''

prints only your username.

## Notes

- You can use id to look up other users.
- root is superuser; system admin
- typing man in front of a command in Linux displays the manual page for that command.
